# Visit Japan — Sample Tourist Website

This is a simple static website scaffold for tourists planning a trip to Japan.

Files
- index.html — main page
- css/style.css — styles
- js/main.js — simple client JS

To run
1. Open `index.html` in a browser (double-click or `open index.html`).

Customize
- Replace photos in `index.html` with your own images or local files.
- Edit itinerary, destinations and form fields to match your tours.

Notes
- This is a static demo. For real bookings integrate a backend or use a service (Formspree, Netlify Forms).
- Map uses a simple Google Maps embed query.
